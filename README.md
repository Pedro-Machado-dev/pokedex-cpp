## How to run

1. Clone this repository:

```bash
git clone https://github.com/Pedro-Machado-dev/pokedex-cpp.git
```

2. Open the project folder:

```bash
cd pokedex-cpp
```

3. Compile the project:

```bash
g++ main.cpp -o pokedex
```

4. Run the program:

```bash
./pokedex
```

On Windows, run:

```bash
pokedex.exe
```


# C++ Pokédex

Academic C++ project developed to manage Pokémon, cities and routes using data structures such as binary search trees, adjacency lists and graph traversal.

## Features

- Register, list and remove Pokémon
- Organize Pokémon by name and type using binary search trees
- Register cities and roads
- Search for the nearest Pokémon Center using graph traversal
- Count Pokémon by type
- Find nearby Pokémon based on coordinates

## Technologies

- C++
- Binary Search Tree
- Graphs
- Adjacency Lists
- Data Structures

## About the project

This project was developed as an academic group assignment for the Software Engineering course.
